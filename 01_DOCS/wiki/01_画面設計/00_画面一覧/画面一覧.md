# 画面一覧
- 画面IDは、画面を識別するための一意の文字列です。コード内で画面を切り替える際に使用されます。
- 画面名は、画面の日本語名称です。
- 説明は、画面の内容や役割を簡潔に説明したものです。

### 画面一覧
| NO | 画面ID    | 画面名      | 説明               |
|:--:|:--------|:---------|:-----------------|
|    | HOME    | ホーム画面    | TOP画面            |
|    | SAVE    | セーブ画面    | セーブ、ロード管理画面      |
|    | CONFIG  | 設定画面     | キーコンフィグや音量設定する画面 |
|    | DUNGEON | ダンジョン画面  | 実際のゲーム画面         |
|    | END     | エンディング画面 | ゲームクリア後の画面       |
