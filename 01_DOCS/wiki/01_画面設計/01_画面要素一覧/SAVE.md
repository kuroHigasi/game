# セーブ画面要素一覧
- セーブ画面に配置される要素の一覧を以下に示します。
- ステータス遷移有無は、要素がユーザーの操作によって状態が変化するかどうかを示しています。
- ステータス遷移がある要素は、ユーザーの操作に応じて異なる状態を持つことができます。例えば、ロードボタンはセーブスロットにデータが存在する場合と存在しない場合で異なる状態を持ちます。
- ステータス遷移がない要素は、常に同じ状態を保ちます。例えば、セーブスロットのカードは、ユーザーの操作に関係なく常に同じ見た目をしています。
- 以下の表は、セーブ画面に配置される要素のID、種別、ステータス遷移の有無、および名称を示しています。

|    | SAVE_BACK_BUTTON     | ボタン | <strong>有</strong> | 前画面への戻る処理        |
|    | SAVE_1_CARD          | カード | 無                  | セーブスロット1         |
