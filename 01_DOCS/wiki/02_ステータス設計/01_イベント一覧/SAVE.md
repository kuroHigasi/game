# SAVEステータスにおける遷移イベント
- SAVEステータスにおいて、ユーザーの操作によって引き起こされる遷移イベントの一覧を示します。
- イベントIDは、イベントを識別するための一意のIDです。
- 画面要素IDは、イベントが発生する画面要素のIDを示します。
- 遷移先は、イベントが発生した際の遷移先のステータスを示します。固定の場合は特定のステータスに遷移し、可変の場合は状況に応じて異なるステータスに遷移することがあります。
- 説明は、イベントの内容や遷移の理由を簡潔に説明したものです。

### SAVEステータスの遷移イベント一覧
| NO | イベントID            | 画面要素ID             | 遷移先（固定/可変） | 説明                      |
|:--:|:------------------|:-------------------|:-----------|:------------------------|
|    | SAVE_BACK_EVENT   | SAVE_BACK_BUTTON   | 可変         | 前画面に戻る                  |
|    | SAVE_1_LOAD_EVENT | SAVE_1_LOAD_BUTTON | 固定         | セーブスロット1からロードしたゲームを開始する |
|    | SAVE_2_LOAD_EVENT | SAVE_2_LOAD_BUTTON | 固定         | セーブスロット2からロードしたゲームを開始する |
|    | SAVE_3_LOAD_EVENT | SAVE_3_LOAD_BUTTON | 固定         | セーブスロット3からロードしたゲームを開始する |

> **注**: セーブスロットにデータが存在しない場合、ロードイベントは発生せず、ユーザーは前画面に戻ることになります。
