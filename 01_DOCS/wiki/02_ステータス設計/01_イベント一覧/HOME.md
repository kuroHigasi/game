# HOMEステータスにおける遷移イベント
- HOMEステータスにおいて、ユーザーの操作によって引き起こされる遷移イベントの一覧を示します。
- イベントIDは、イベントを識別するための一意のIDです。
- 画面要素IDは、イベントが発生する画面要素のIDを示します。
- 遷移先は、イベントが発生した際の遷移先のステータスを示します。固定の場合は特定のステータスに遷移し、可変の場合は状況に応じて異なるステータスに遷移することがあります。
- 説明は、イベントの内容や遷移の理由を簡潔に説明したものです。

### HOMEステータスの遷移イベント一覧
| NO | イベントID            | 画面要素ID             | 遷移先（固定/可変） | 説明                     |
|:--:|:------------------|:-------------------|:-----------|:-----------------------|
|    | HOME_START_EVENT  | HOME_START_BUTTON  | 固定         | ゲーム開始                  |
|    | HOME_SAVE_EVENT   | HOME_SAVE_BUTTON   | 固定         | データをロードするためにセーブスロットを確認 |
|    | HOME_CONFIG_EVENT | HOME_CONFIG_BUTTON | 固定         | 操作や音量を調整する             |
|    | HOME_EXIT_EVENT   | HOME_EXIT_BUTTON   | 固定         | ゲーム終了                  |
