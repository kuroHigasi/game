# CONFIGステータスにおける遷移イベント

| NO | イベントID            | 画面要素ID             | 遷移先（固定/可変） | 説明                 |
|:-:|:------------------|:-------------------|:-----------|:-------------------|
|    | CONFIG_OK_EVENT   | CONFIG_OK_BUTTON   | 可変         | 設定を反映後、前のステータスに戻る  |
|    | CONFIG_BACK_EVENT | CONFIG_BACK_BUTTON | 可変         | 設定を反映せず、前のステータスに戻る |
