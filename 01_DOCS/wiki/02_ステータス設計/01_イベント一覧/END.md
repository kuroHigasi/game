# ENDステータスにおける遷移イベント

| NO | イベントID          | 画面要素ID        | 遷移先（固定/可変） | 説明        |
|:-:|:----------------|:--------------|:-----------|:----------|
|    | END_TITLE_EVENT | END_TITLE_BTN | 固定         | タイトルに戻る   |
|    | END_EXIT_EVENT  | END_EXIT_BTN  | 固定         | アプリケーション終了 |

> エンディング画面では、プレイヤーはタイトル画面に戻るか、ゲームを終了するかを選択できます。
