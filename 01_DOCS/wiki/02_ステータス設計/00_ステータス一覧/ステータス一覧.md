# ステータス一覧
ステータスは、ゲーム内の様々な画面や状態を識別するためのものです。以下に、各ステータスのID、適用される画面名、および説明を示します。
- ステータスID: ステータスを識別するためのID
- 画面名: ステータスが適用される画面の名前
- 説明: ステータスの説明

| ステータスID | 画面名      | 説明               |
|:--------|:---------|:-----------------|
| HOME    | ホーム画面    | TOP画面            |
| SAVE    | セーブ画面    | セーブ、ロード管理画面      |
| CONFIG  | 設定画面     | キーコンフィグや音量設定する画面 |
| DUNGEON | ダンジョン画面  | 実際のゲーム画面         |
| END     | エンディング画面 | ゲームクリア後の画面       |

> 基本的に画面一覧の画面IDと同一のステータスIDを使用しますが、必要に応じて画面内の特定の状態を識別するために、追加のステータスIDを定義することもあります。
