# ステータス遷移一覧
- ステータス遷移の一覧を以下に示します。
- 各イベントIDは、ゲーム内で発生する特定のアクションや条件に対応しています。
- ステータス遷移は、ユーザーの操作やゲームの進行に応じて、画面や状態がどのように変化するかを示しています。
- 以下の表は、遷移元のステータス、イベントID、遷移先のステータス、および各遷移の説明をまとめたものです。
- この一覧を参考に、ゲームのフローやユーザーの操作に対する反応を理解できます。

### ステータス遷移一覧
| NO | ステータス<br>（遷移元） | イベントID            | ステータス<br>（遷移先） | 説明               |
|:--:|:---------------|:------------------|:---------------|:-----------------|
|    | HOME           | HOME_START_EVENT  | DUNGEON        | ゲーム開始            |
|    | HOME           | HOME_SAVE_EVENT   | SAVE           | セーブスロットを確認       |
|    | HOME           | HOME_CONFIG_EVENT | CONFIG         | 設定を確認            |
|    | HOME           | HOME_EXIT_EVENT   | EXIT           | ゲーム終了            |
|    | CONFIG         | CONFIG_OK_EVENT   | HOME           | 設定を反映            |
|    | CONFIG         | CONFIG_OK_EVENT   | DUNGEON        | 設定を反映            |
|    | CONFIG         | CONFIG_BACK_EVENT | HOME           | 設定をそのまま戻る        |
|    | CONFIG         | CONFIG_BACK_EVENT | DUNGEON        | 設定をそのまま戻る        |
|    | SAVE           | SAVE_BACK_EVENT   | HOME           | ゲームをロードせず戻る      |
|    | SAVE           | SAVE_BACK_EVENT   | DUNGEON        | ゲームをロードせず戻る      |
|    | SAVE           | SAVE_1_LOAD_EVENT | DUNGEON        | セーブスロット1のデータをロード |
|    | SAVE           | SAVE_2_LOAD_EVENT | DUNGEON        | セーブスロット2のデータをロード |
|    | SAVE           | SAVE_3_LOAD_EVENT | DUNGEON        | セーブスロット3のデータをロード |
|    | DUNGEON        | DUNGEON_END_EVENT | END            | ゲームクリア           |
|    | END            | END_TITLE_EVENT   | HOME           | タイトルに戻る          |
|    | END            | END_EXIT_EVENT    | EXIT           | ゲーム終了            |

> **注**: セーブスロットにデータが存在しない場合、ロードイベントは発生せず、ユーザーは前画面に戻ることになります。
